# 心理疲惫度测试 · 个性化报告话术库

## 一、总评模板（按疲惫等级）

### 1.0级 · 安全区
```
你现在处于【0级 · 安全区】，整体的情绪状态相对稳定，没有明显的疲惫积累。

从测评结果看，你目前没有严重的情绪消耗，恢复资源相对充足。这是一个很好的基础状态，建议你保持现有的自我觉察习惯，定期关注自己的情绪变化。

建议关注：如何在这个基础上建立更强的心理韧性，为未来可能的压力挑战做准备。
```

### 1.1级 · 轻度疲惫
```
你目前处于【1级 · 轻度疲惫】，可能最近有一些小事让你觉得有点心累，但整体还在可控范围内。

这种状态很多人都会经历，通常是因为某些场景或压力开始悄悄消耗你的心理能量。好消息是，你及时发现并关注到了这些信号，这是很好的自我觉察。

轻度疲惫是一个提醒：需要开始注意自己的节奏，适当调整，避免消耗的进一步积累。
```

### 1.2级 · 中度疲惫
```
你目前处于【2级 · 中度疲惫】，说明你的情绪能量已经有明显的消耗，可能经常感到疲惫、提不起精神。

这个程度已经不是"休息一下就能好"的程度了，而是需要主动关注和调整的状态。你的疲惫感可能来自于特定的场景压力，或者是长期的某些行为模式导致的累积。

中度疲惫是一个重要的转折点：如果及时调整，可以有效避免进入高度疲惫的状态；如果继续忽视，可能会进一步恶化。
```

### 1.3级 · 高度疲惫
```
你目前处于【3级 · 高度疲惫】，这表明你的情绪和心理资源已经被严重消耗，长期处于"硬撑"的状态。

从测评结果看，你的疲惫已经不仅仅是累，而是深层次的心理耗竭。可能的表现包括：对很多事情失去兴趣、情绪容易波动、即使休息也感觉恢复不过来、经常感到力不从心。

高度疲惫是一个需要高度重视的信号：它意味着你需要立即采取行动，不能再继续"扛着走"，否则可能会影响到身心健康。
```

### 1.4级 · 极度疲惫
```
你目前处于【4级 · 极度疲惫】，这是情绪耗竭的最高等级，说明你的心理资源已经严重透支，可能已经影响到日常生活和正常功能。

在这个状态下，你可能感到：对什么都提不起兴趣、记忆力注意力下降、情绪极度不稳定、身体也出现各种不适症状。这不是"矫情"或"软弱"，而是长期压力累积导致的真实状态。

极度疲惫是一个紧急信号：强烈建议你寻求专业的心理支持，同时立即调整生活节奏。这不是你一个人的问题，而是需要外部帮助来走出困境。
```

## 二、主类型分析模板（4套完整模板）

### 2.1 情绪过载型（主类型）
```
【情绪过载型】的核心特征是高度敏感和强烈共情，容易吸收他人的情绪能量。

**典型心态模式：**
你对他人的情绪变化特别敏感，能够敏锐捕捉到微小的氛围变化。这种敏感性让你很容易理解别人，但同时也让你过度承担他人的情绪负担。当身边有人不开心时，你会不自觉地想要"修复"这种状况，把别人的焦虑当作自己的问题。

**常见行为模式：**
- 经常在意别人的反应，担心自己的言行让他人不快
- 容易被他人的负面情绪影响，一整天心情都会被改变
- 习惯性地站在他人角度考虑问题，而忽略自己的需求
- 在冲突或不和谐的氛围中感到极度不适，想要赶紧调解

**容易掉入的陷阱：**
1. **情绪边界模糊**：很难区分哪些是自己的情绪，哪些是吸收来的情绪
2. **过度责任化**：把别人的情绪状态当作自己的责任
3. **自我牺牲**：为了维护表面和谐，过度压抑自己的真实感受
4. **情绪透支**：不断地为他人"充电"，而自己的情绪电池却越来越弱

这种类型的人往往被认为是"贴心""善解人意"的，但这种品质如果缺乏边界，就会成为持续消耗的源头。
```

### 2.2 责任绑架型（主类型）
```
【责任绑架型】的核心特征是强烈的责任感和难以拒绝他人的需求，习惯性地承担超出自己能力范围的事情。

**典型心态模式：**
你内心深处有一个声音："如果我不做，事情就会出问题。"这种心态让你很难对别人的请求说"不"，也很难真正把事情交给他人。你担心自己拒绝别人会被认为是自私，或者害怕事情没有"你"的参与就会搞砸。

**常见行为模式：**
- 团队中总是扮演"兜底者"的角色，最后收尾的总是你
- 即使已经很累了，还是会答应额外的任务或请求
- 难以接受不完美的结果，总觉得"自己再多做一点会更好"
- 常常说"我来吧"，因为这样比解释或等待更有效率

**容易掉入的陷阱：**
1. **万能救世主情节**：认为没有自己就不行，高估自己的不可替代性
2. **拒绝困难症**：说"不"会产生强烈的愧疚感和自我怀疑
3. **控制焦虑**：担心事情不在自己掌控范围内会出问题
4. **身份认同依赖**：过度依赖"负责任""可靠"的他人评价来定义自我价值

这种类型的人往往被认为是"可靠""能干"的，但这种责任感如果缺乏边界感，就会变成沉重的负担。
```

### 2.3 对比焦虑型（主类型）
```
【对比焦虑型】的核心特征是容易产生社会比较焦虑，过度关注他人的成就和表现，并以此作为自我评价的标准。

**典型心态模式：**
你的自我评价系统高度依赖外部参照系。看到同龄人的成功、收入、生活状态时，内心会产生强烈的比较压力。即使理智上知道每个人的路径不同，但情感上还是忍不住拿来和自己比较，然后产生"为什么我不行"的焦虑。

**常见行为模式：**
- 经常浏览社交媒体，了解别人的"成功故事"然后感到压力
- 会给自己设定"这个年龄应该达到什么"的时间表和目标
- 听到别人的表扬或批评时，内心波动很大
- 很难真正为自己的成就感到满足，总觉得"还不够"

**容易掉入的陷阱：**
1. **时间焦虑**：过度担心自己"落后"于同龄人或预期的时间表
2. **成就标准外移**：用自己的劣势去对比他人的优势，产生不公平的自我评价
3. **永不满足**：总有一个"下一个目标"，很难真正享受当下的成就
4. **独特性迷失**：为了符合社会期待，忽略自己真正想要和擅长的事情

这种类型的人往往很有上进心，但如果过度依赖外部比较，就会陷入永远无法满足的焦虑循环中。
```

### 2.4 高压稳定型（主类型）
```
【高压稳定型】的核心特征是习惯在高强度、高节奏的状态下运作，把"忙碌"等同于"有价值"，难以适应慢节奏的休息状态。

**典型心态模式：**
你已经适应了持续高压的运转模式，当突然有很多空闲时间时反而会感到不安和不适应。你内心深处相信"停下来就是落后""忙碌意味着重要"，因此很难真正允许自己休息。即使身体发出疲惫信号，你也倾向于"咬牙撑过去"而不是"先调整自己"。

**常见行为模式：**
- 习惯同时处理多个任务，享受"高效运转"的感觉
- 即使已经很累了，还是会给自己安排新的事情做
- 很难向他人承认自己"累了"或"需要帮助"，觉得这是软弱的表现
- 把"我能扛"当作自我证明，认为稳定输出就是专业素养

**容易掉入的陷阱：**
1. **速度成瘾**：把快节奏当作常态，慢下来反而感到空虚和焦虑
2. **脆弱的稳定感**：所谓的"稳定"实际上是持续绷紧的状态，一旦遇到意外就容易崩溃
3. **自我忽视**：过度关注外部任务，忽略身体和心理的疲惫信号
4. **求助羞耻感**：认为承认需要帮助等于承认能力不足

这种类型的人往往被认为是"高效""稳定"的，但这种稳定感如果建立在持续透支的基础上，其实是非常脆弱的。
```

## 三、副类型补充模板（4套简短版本）

```
这种特质让你在团队中很有共情力，但也可能让你成为团队的"情绪吸收器"，在不自觉中承担了超出自己职责范围的情绪负担。

建议：在保持同理心的同时，学会设置情绪边界，识别哪些是自己的情绪，哪些是来自他人的情绪投射。
```

```
这种让你在别人眼中很"可靠"，但也可能让你成为团队中的"万能钥匙"，什么事都找你，逐渐超出自己的承载能力。
```

```
这种关注外部的特质让你有很强的上进心，但也可能让你忽略了自己的节奏和优势，过于关注别人的评价标准。
```

```
这种应对策略短期内有效，但长期来看，可能会让你忽略了必要的休息和调整，把"硬扛"当作解决问题的唯一方式。
```

## 四、场景分析模板（按场景和强度）

### 4.1 工作/学习场景（高/中/低强度）

#### 高强度（分数≥3.5）
```
【工作/学习】是你目前最主要的"掉血源"。测评显示，你在工作/学习场景中承受着持续的压力和消耗。

这种高强度的工作压力可能表现为：
- 经常感到"任务永远做不完"的焦虑
- 对自己的表现过度担忧，害怕被否定或淘汰
- 即使很累了，也不敢拒绝新的任务
- 很难从工作中获得成就感和满足感

这种状态往往是长期累积的结果：一开始可能是高强度项目阶段，但如果没有及时调整，就会变成持续的消耗模式。当工作压力超过了你的恢复能力时，就会出现明显的疲惫症状。

需要警惕的是：工作场景的高强度往往会"传染"到其他生活领域，让你即使在休息时也难以真正放松。
```

#### 中等强度（分数2.5-3.4）
```
【工作/学习】对你来说有一定的压力，但还在可管理范围内。测评显示，你在工作/学习中感受到了一定的消耗，但没有达到严重的程度。

这种中等强度的工作压力可能意味着：
- 偶尔会感到工作量大，但总体还能应对
- 有时会担心自己的表现，但不会过度焦虑
- 在拒绝额外工作时会感到一些愧疚，但还是能够设置边界
- 大部分时候还能从工作中获得一定的满足感

这种状态建议及时关注和调整：虽然目前还不是严重问题，但如果持续增加压力，可能会逐渐变成主要消耗源。建议你主动管理工作节奏，预防压力的进一步累积。

关键在于建立可持续的工作模式，而不是等到问题严重了才被迫调整。
```

#### 低强度（分数<2.5）
```
【工作/学习】目前对你来说不是主要的压力源。测评显示，你在工作/学习场景中的压力相对可控，没有造成严重的消耗。

这表明：
- 你在工作/学习中的压力管理相对有效
- 能够比较自然地设置工作边界
- 大部分时候能从工作中获得正面体验
- 工作压力没有明显溢出到个人生活

这是一个很好的状态，建议你保持现有的工作边界和节奏。不过也要注意，工作场景的低压力并不意味着可以忽视其他可能的压力来源。

保持这种健康的工作状态需要持续的自我觉察和边界维护。
```

```
【人际/家庭关系】对你来说有一定的压力，但总体还在平衡范围内。测评显示，你在人际交往中感受到了一定的负担，但没有达到严重的程度。

这种中等强度的人际压力可能意味着：
```

```
【人际/家庭关系】目前对你来说不是主要的压力源。测评显示，你在人际交往中的消耗相对较低，关系总体是健康的。

这表明：

这是一个很健康的人际状态。健康的边界不仅保护了你，也让关系更加可持续。建议你继续保持这种人际边界意识，同时也要警惕在某些特殊情况下（如家人遇到困难时）边界被模糊的可能。

良好的人际关系质量是心理健康的重要保护因素，继续保持。
```

### 4.3 自我要求/内心压力场景（高/中/低强度）

#### 高强度（分数≥3.5）
```
【自我要求/内心压力】是你目前最大的内在消耗源。测评显示，你对自己的标准和要求极高，这造成了持续的内在压力。

这种高强度的自我要求可能表现为：
- 对自己有很多"应该做到"的标准，一旦没做到就会强烈自责
- 当稍微休息或放松时，内心会有"浪费时间""不够努力"的声音
- 很难接受自己状态不好或表现一般，总觉得"我应该能再好一点"
- 即使别人觉得你已经做得够多了，但自己仍然觉得"不够、不够、不够"

内在的自我要求是最难应对的压力源，因为它来自内心，无法通过逃避来解决。这种高度的自我驱动力在某些方面可能是优势，但如果过度，就会变成持续的自我苛责和内在冲突。

需要警惕的是：过高的自我要求往往会让你忽视自己的真实需求和感受，长期来看会严重影响身心健康和幸福感。
```

#### 中等强度（分数2.5-3.4）
```
【自我要求/内心压力】对你来说有一定的影响，但还没有达到严重的程度。测评显示，你对自己的要求相对较高，但有一定的自我调节能力。

这种中等强度的自我要求可能意味着：
- 对自己有一定的标准和期待，但偶尔也能接受不完美
- 有时会因为放松而内疚，但总体还能享受休息的时光
- 在表现不理想时会自责，但通常能够客观看待
- 有时会觉得"自己还能做得更好"，但不会过度强迫自己

这种状态建议你关注自我接纳：虽然适度的自我要求有助于成长，但过度的自我苛责会成为持续的内耗源。建议你学会区分"健康的自我要求"和"过度的自我苛责"。

关键在于培养更温和的自我对话方式，学会用对自己更友善的眼光看待自己的表现和状态。
```

#### 低强度（分数<2.5）
```

- 看似稳定的表象下隐藏着崩溃的风险

这种模式的警示：
- 当前的"稳定"是建立在持续透支的基础上，不可持续
- 身体和心理的疲惫信号在不断增加，但被忽视
- 一旦遇到额外的压力或挑战，可能突然崩溃

**调整重点：**
1. 重新定义"稳定"，追求可持续的平衡而非持续的高压
2. 学会识别和重视身体的疲惫信号，及时调整节奏
3. 重建恢复力，培养真正有效的休息和放松技巧
4. 逐步降低生活节奏，适应更健康的工作生活平衡
```

### 9.2 报告结构组织
```typescript
interface ReportStructure {
  // 1. 总评部分（1段）
  overview: string;

  primaryTypeAnalysis: string;

  secondaryTypeSupplement?: string;

  };
```

### 9.3 避免模板爆炸的管理策略

1. **模板复用设计**：
   - 使用占位符实现模板的灵活复用
   - 例如：场景模板可以用同一个结构，替换具体场景名称

2. **模块化组合**：
   - 每个模板片段都是独立的，可以灵活组合
   - 避免编写大量完整的长文档

3. **渐进式丰富**：
   - 先实现基础版本，后续逐步增加更多模板片段
   - 根据用户反馈不断优化和丰富

4. **版本化管理**：
   - 为不同版本的话术建立版本控制系统
   - 支持A/B测试不同模板的效果

通过这套完整的话术体系，可以为每个用户生成高度个性化、有深度的情绪疲惫分析报告，既保证了专业性，又实现了真正意义上的个性化体验。

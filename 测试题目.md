模块 A：整体疲惫强度（8 题）

用来判断「你到底有多累」，不区分场景、不谈来源。

A1 最近一段时间，我早上醒来时，经常觉得自己根本没有真正休息好。
A2 明明睡眠时间差不多，但整个人总是提不起精神、像手机电量一直不满格。
A3 一天下来，我会觉得脑子“被掏空”，哪怕什么都不做也很难真正放松。
A4 最近，我很容易对小事失去耐心，情绪一下子就被点燃或拉低。
A5 我经常需要强迫自己去完成本来应该轻松完成的事情。
A6 我常常会有一种说不出原因的心累，明明没有发生什么大事，却总觉得压得喘不过气。
A7 最近，我的注意力很难集中，看东西、开会或聊天都会不自觉走神。
A8 即使有空闲时间，我也很难真正投入一件事，总有一种“怎么都休息不够”的感觉。

模块 B：疲惫场景 / 掉血地图（12 题）

回答「你主要是在哪些场景被榨干」：
子维度：

B1–B4：工作 / 学习

B5–B8：人际 / 家庭

B9–B12：自我要求 / 内心对话

B.1 工作 / 学习压力（4 题）

B1 面对工作 / 学习任务时，我经常有一种“怎么追也追不完”的焦虑感。
B2 我常常担心自己在工作 / 学习上的表现不够好，会被否定或被淘汰。
B3 即使已经很累了，我还是会不自觉地多接一些工作 / 任务，生怕被说不上进。
B4 最近，我在工作 / 学习中很难找到成就感，更多是为了“撑过去”“别出事”而硬撑。

B.2 人际 / 家庭关系消耗（4 题）

B5 我会为了不让别人失望或生气，答应很多自己其实不太想做的事。
B6 和家人、伴侣、朋友相处时，我经常在意他们的情绪，生怕自己说错话或做错事。
B7 有时我会觉得自己像“情绪垃圾桶”，别人把负面情绪倾倒给我后，我还得自己消化。
B8 和身边的人相处后，比起轻松，我更容易感觉到被消耗、被掏空。

B.3 自我要求 / 内心压力（4 题）

B9 我对自己有很多“应该做到”的标准，一旦没做到就会强烈自责。
B10 当我稍微休息一下或偷个懒时，心里会冒出“这样很没用、很浪费时间”的声音。
B11 我很难接受自己状态不好或者表现一般，总觉得“我应该能再好一点”。
B12 即使周围人已经觉得我做得够多了，我自己仍然觉得“不够、不够、不够”。

模块 C：疲惫来源类型（16 题）

回答「我为什么会这么累」——主类型 + 副标签就从这里算出来。
每种类型 4 题，你后端可以按 group 做平均分 → 映射为匹配度。

子类型代码建议（方便开发时标注）：

情绪过载型：type_emotional_overload

责任绑架型：type_responsibility_trap

对比焦虑型：type_comparison_anxiety

高压稳定型：type_high_pressure_mode

C.1 情绪过载型（4 题）

type_emotional_overload

C1 周围人的情绪变化（比如语气微妙变化、表情不对）很容易被我捕捉到，也会影响我一整天的心情。
C2 别人遇到困难或心情不好时，我会下意识把他们的感受揽到自己身上，替对方着急、替对方难受。
C3 即使事情已经过去了，我还是会反复回想当时的对话和细节，揣测自己有没有哪里做得不好。
C4 面对冲突或不开心的场面时，我比一般人更容易紧张、心慌，很想赶紧把气氛“修好”。

C.2 责任绑架型（4 题）

type_responsibility_trap

C5 很多时候我会觉得：“这件事如果我不管，就会出问题”，于是明明很累也会硬着头皮上。
C6 当我拒绝别人、说“不方便”或“不行”时，会感到强烈的愧疚，好像自己是个自私的人。
C7 我经常接手别人本可以自己处理的事情，只因为“不想麻烦对方”或“不想看他们手忙脚乱”。
C8 我习惯在团队 / 家庭里扮演“兜底”的角色，出事先看我、收尾也找我。

C.3 对比焦虑型（4 题）

type_comparison_anxiety

C9 刷社交媒体或听到同龄人的近况时，我很容易不自觉地和他们比较，感觉自己落后了。
C10 当别人分享自己的成绩、收入或生活时，我表面上很平静，内心却会暗暗给自己很大压力。
C11 我常常会给自己设定一些“这个年龄应该达到什么水平”的目标，一旦没达到就很焦虑。
C12 听到别人说“谁谁做得很好”“谁谁很厉害”时，我会立刻联想到自己是不是不够好。

C.4 高压稳定型（4 题）

type_high_pressure_mode

C13 我已经习惯在高强度、高节奏下工作 / 生活，一旦突然闲下来反而会不安。
C14 遇到任务或问题时，我的第一反应通常是“咬牙撑过去”，而不是“我能不能先调整一下自己”。
C15 即使身体已经出现一些疲惫信号（比如失眠、头痛、心悸等），我也会安慰自己“再撑一阵就好了”。
C16 我很难向别人坦白自己累了、扛不住了，总觉得那样会显得自己不够坚强或不够专业。

模块 D：恢复资源 / 保护因子（4 题）

回答「我有没有机会好起来、手上还有多少‘回复血量’的筹码」。

D1 最近一段时间，我整体的睡眠质量还算可以，能在大部分夜晚获得相对完整的休息。
D2 我的生活里有一两个可以真正说心里话的人，和他们聊完会感觉轻松一些。
D3 我偶尔会给自己留出「只属于自己、不需要对任何人负责」的时间，用来放空或做喜欢的事情。
D4 当我意识到自己真的撑不住时，我有能力也敢于说“我需要暂停一下 / 需要帮助”。
